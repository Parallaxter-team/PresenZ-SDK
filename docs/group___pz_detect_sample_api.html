Detailed Description

Contains the interface for the PresenZ probe casting, this interface is used in the detect phase to.

calculate ray intersections with objects in the scene.These intersections also return information on the type of object that was hit. For instance, whether hair or glass was hit in the scene. In the case a transparent object is hit (tagged with a "prz_transparent" tag), probe casting will ignore it and continue the detection ray until it either hits an opaque object or reaches the maximum allowed rendering distance. The interface is mostly concerned with detecting which parts of the scene are visble from within the Zone Of View, we gather geometry information and forward these to PresenZ.
