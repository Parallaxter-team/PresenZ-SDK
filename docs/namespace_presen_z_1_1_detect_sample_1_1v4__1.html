◆ PzDetectSwap()

void PzDetectSwap ( )

In order to improve efficiency, we are trying to parallel the CPU and GPU jobs.

While OptiX performs ray tracing test in GPU, PresenZ is proceeding data in CPU. Therefore, there are two data sets inside PresenZ SDK. While one data set is proceeded in PresenZ SDK, the other data set is transferred to OptiX for ray tracing test. Then, the two data sets are swapped for next round of proceeding.
