Detailed Description

Contains the shading interface for PresenZ. This interface is used in the Render Phase to process transparent rays and bend rays for the left and right eye.

Ray Bending

Rays are typically emitted from a camera's point of view (more specifically from it's world location). To create a stereoscopic render we need to bend the rays to line up with the view of the left and right eye.
